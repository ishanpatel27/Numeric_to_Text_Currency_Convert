Console application that takes in a currency value (Min Value 00, Max Value 9,99,999.99) and prints out a text.

# Author : Ishan Patel

------------------------------------------------------
Project Directory Structure :

-	.coverage
-	currconvert.py  # Main Code
-	htmlcov		# Folder contains Coverage Report in Html format 
-	readme.txt
-	test_c.py	# Unit Test File
-	__pycache__

-------------------------------------------------------
Prerequisites :

Python 3.8
-------------------------------------------------------
Steps to Run the Program :

1. Run " currconvert.py "  file in Python Console.
2. Enter the currency when prompted in numeric form ex : 198   
3. Hit Enter .
4. Output will be printed on console.
